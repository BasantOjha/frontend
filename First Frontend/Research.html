HTML

HTML stands for Hyper Text Markup Language that is used to create the basic structure of the webpage, to form the skeleton of the webpage. It is not a programming language it is a markup language that is used to give structure to the text document and uses tags to define an element in a document. Browser does not display the tags of HTML but display the content that are written within the tags. HTML relies on other technologies such as CSS (Cascading Style Sheets) and JavaScript as the tools to make websites look really colorful attractive and responsive.

CSS

CSS stands for Cascading Style Sheet that is used to describe and change the style of the HTML document. It helps to describe how the html document should be displayed. CSS define for the style of a particular HTML document or element once can be used over and over on any of the web pages. CSS style rule consists of two parts that are selector and deceleration where selector can be a html tag and deceleration is the format (width, height, color, font, etc.) that u want for the elements and documents present inside that tag.

There are three types of CSS

  • Internal CSS
  • External CSS
  • Inline CSS

JavaScript

JavaScript is a light weight programming language that is used in HTML to make webpage look interactive and also to insert dynamic text into the HTML that ususally react to events. JavaScript is not related to Java other than by name and by some syntactic similarities. It is interpreted language. JavaScript code runs in the browser and is case sensitive and can change the content of HTML elements and CSS properties easily.
